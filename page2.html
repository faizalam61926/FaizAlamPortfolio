Project Revenge

About

Project Revenge is an immersive VR hack-and-slash melee combat game set in the Medieval period. The game features a combat system based on a token system, similar to those used in AAA action titles like Marvel's Spider-Man and God of War.

Project Info

Role: Programmer, Animation
Team Size: 4
Time frame: 3 months
Engine: Unreal Engine
Year: 2021

Introduction:
The game was an experiment my team was doing to make a good feeling VR combat similar to god of war and Asgard's wrath

Token System Combat
We made a token system combat similar to what God of war and 2018 Spiderman had and translated that to VR
it uses combination of EQS and Behaviour tree AI systems of Unreal Engine

Team Management:
We had 4 members working on it, 2 programmers, 1 environment artist, 1 character artist

Motion Capture:
I made Character animations using Iclone 8 and Perception neuron's Motion capture suit and transferred the data to Unreal Engine to create high quality attack and hit reaction animations
