Ultimate Zombie Shooter

About

Ultimate Zombie Shooter is an old-school on-rails shooter inspired by classic titles like House of the Dead and Virtua Cop. Developed using Unreal Engine, the game features lightgun functionality for an authentic arcade shooting experience. Additionally, a swipe card system implemented with Arduino enhances the gameplay by providing seamless access and user interaction.

Project Info

Role: solo
Team Size: 1
Time frame: 4 months
Engine: Unreal Engine
Year: 2022

Introduction:
The Experience was made for Smaaash Arcade center, it uses a USB lightgun that acts like a mouse and connected to Unreal Engine, i used Arduino as well which is connected with the card reader that triggers the game to start, it has 3 levels with a boss battle at the end of each level, it was inspired from the games like virtua cop, house of the dead and time crisis.

On Rail Experience
users will follow a pre defined path while shooting the zombies, zombies have AI to move towards the player and attack them, the bosses have their pattern to attack which is centered around the player, the gameplay is designed to make the player feel powerful. 

Live at Locations:
the experience is still live at multiple locations at Smaaash arcade centers.
