Midnight at Blackwood manor

About

It is a story-driven deduction game where players step into the shoes of a detective solving a mysterious murder at a lakeside mansion. The game features a unique deduction system, allowing players to come up with their own theories and solve the case in their own way. With its intricate puzzles, atmospheric exploration, and compelling narrative, Lake House Murder offers a highly immersive and personalized investigative experience.

Project Info

Role: Game Design, Animation
Team Size: 5
Time frame: 8 months
Engine: Unreal Engine
Year: 2023- 2024

Introduction:
The game is inspired by detective games like Agatha Christie's games with unique deduction system designed from scratch to set this game apart from other detective games in the market.

Dialogue system
Designed dialogue system using data tables and data assets in Unreal engine, selected dialogue has corresponding response in the data table stored and can be changed or swapped at any time in one place giving developers easy access to change story lines and improve dialogue lines when needed

Robust Deduction system
Designed a deduction system which allows user input, it takes place on a 3D UI where users can place notes, photos, evidence and link them together to create possible outcomes and generate questions to ask the suspects.
